# 🚩 HackTheFlag Writeups

A collection of my CTF writeups for the **Hack the Flag** cybersecurity competition hosted at [react.ct.ws](http://react.ct.ws).

---

## 📁 Categories

| Category | Description |
|----------|-------------|
| 🛠️ General_Skills | Basic skills and tools |
| 🌐 Web_Exploitation | Web vulnerabilities and attacks |
| 🔐 Cryptography | Ciphers and encoding challenges |
| 🔍 Forensics | File analysis and data recovery |
| ⚙️ Reverse_Engineering | Code analysis and reversing |
| 💥 Binary_Exploitation | Memory and binary attacks |
| 🕵️ OSINT | Open source intelligence gathering |
| 🎲 Miscellaneous | Everything else |

---

## 📝 Writeup Format

Each writeup includes:
- Challenge description and hints
- Step-by-step solution
- Why it works
- Alternative methods
- Tools used
- Key takeaways
